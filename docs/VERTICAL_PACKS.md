# Verticales Iniciales

## Principio

Jarvis debe aprender verticales con rigor. Un vertical pack no es una lista generica de consejos; es una base operativa para entender un tipo de negocio.

## Primeros verticales sugeridos

### Dentistas

Necesidades:

- Agenda.
- Recordatorios de pacientes.
- Preparacion de reuniones.
- Gestion de reseñas.
- Contenido educativo.
- SEO local.
- Documentos administrativos.
- Seguimiento comercial.

Fuentes preferidas:

- Sitios oficiales de salud.
- Colegios profesionales.
- Normativas locales.
- Fuentes academicas.
- Documentacion de software dental cuando aplique.

Riesgos:

- Informacion medica sensible.
- Consejos clinicos.
- Datos de pacientes.
- Cumplimiento legal y confidencialidad.

### Restaurantes

Necesidades:

- Reservas.
- Menu.
- Inventario.
- Reseñas.
- Redes sociales.
- Proveedores.
- Costos.
- Promociones.
- SEO local.

Fuentes preferidas:

- Fuentes gastronomicas profesionales.
- Normativa sanitaria local.
- Tendencias verificadas.
- Competencia local real.
- Plataformas de reseñas.

Riesgos:

- Informacion desactualizada de precios.
- Normativa sanitaria local.
- Automatizaciones que afecten reservas o pagos.

## Criterios de calidad

Cada pack debe incluir:

- Fuentes permitidas.
- Fuentes prohibidas o debiles.
- Preguntas iniciales al cliente.
- Tareas tipicas.
- Riesgos del rubro.
- Plantillas utiles.
- Skills recomendadas.
- Conectores probables.
- Metricas de exito.
